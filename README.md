## Quick Start  


Installation is done using the git clone :
```console
git clone https://github.com/TeamEarendel/SGSS.git
```

## Features  

* Vector2 from Unity Magnitude
* Vector2 from Unity Normalized
* Vector2 from Unity Distance
* Auto Connection Server
